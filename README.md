# CTC Project – PHP Web Application

A full-stack web application built using PHP and MySQL to demonstrate backend development and database integration.

---

## 🚀 Overview

This project was developed to handle user interaction and data processing through a structured backend system.

It combines front-end interfaces with PHP backend logic and MySQL database connectivity.

---

## ✨ Features

* User registration and login system
* Database integration
* Form handling and validation
* Dynamic content rendering

---

## 🛠️ Tech Stack

* PHP
* MySQL
* HTML
* CSS
* JavaScript

---

## ⚙️ How to Run

1. Install XAMPP
2. Place project in `htdocs/`
3. Start Apache & MySQL
4. Import database
5. Open `http://localhost/ctc_project`

---

## 📚 Learning Outcomes

* Backend development with PHP
* Database integration
* Full-stack project structure
