# CTC Project – Web Application

A PHP-based web application developed to demonstrate full-stack development concepts including backend logic, database integration, and user interaction.

---

## 🚀 Overview

The CTC Project is a web application designed to handle user interactions and data processing through a structured backend system.

It focuses on building a functional application using **PHP and MySQL**, combining front-end interfaces with backend processing.

---

## ✨ Features

* User interface for interaction
* Backend logic using PHP
* Database integration for storing and retrieving data
* Form handling and validation
* Structured project layout
* Dynamic content rendering

---

## 🛠️ Tech Stack

* **PHP**
* **MySQL**
* **HTML5**
* **CSS3**
* **JavaScript**

---

## 📂 Project Structure

```text
ctc_project/
│
├── css/
├── js/
├── images/
├── config/          # Database configuration
├── includes/        # Reusable PHP files
├── index.php
├── login.php
├── register.php
└── database.sql
```

---

## ⚙️ How to Run

1. Install **XAMPP** or any local server
2. Place project inside:

   ```
   htdocs/
   ```
3. Start Apache and MySQL
4. Import database:

   * Open phpMyAdmin
   * Import `database.sql`
5. Open in browser:

   ```
   http://localhost/ctc_project
   ```
